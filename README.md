# Web Traffic Analytics

## Objective
Analyze website traffic data to understand user behavior and engagement.

## Tools Used
- Python
- Pandas

## Metrics Analyzed
- Page Views
- Session Duration
- Exit Pages
- User Engagement

## Findings
Users with longer session durations viewed more pages. The Home page and Checkout page were common drop-off points.

## Recommendations
- Improve landing page content
- Optimize checkout experience
- Increase user engagement through better navigation
